## Overview

Spam Checker API offers spam identification and phone lookup services via a mobile app. It includes user registration, secure authentication, search, and spam reporting functionalities.

## Key Features

- **User Registration & Authentication**: Secure sign-up and login with JWT.
- **Search & Spam Reports**: Identify potential spam numbers and report them to help others.
- **Conditional Data Display**: Shows the email only if users are in each otherâ€™s contact lists.

## Technologies

- **Stack**: Node.js, Express, PostgreSQL, Prisma, JWT.
- **Security**: bcryptjs for password hashing.

## Quick Start

- **Seed Database**: `node seed.js`
- **Run App**: `npm run dev` (Server runs on `localhost:3000`)
- **Postman Collection**: Import `spam-checker-api.postman_collection.json` to test API endpoints.

## Outcome

The Spam Checker API provides a powerful tool for spam detection, making it easier for users to manage and avoid unwanted calls.
